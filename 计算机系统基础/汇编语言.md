
# 1. 汇编语言的组成

## 1.1 寄存器（Register）

`寄存器` 是CPU内部的高速存储单元，用于存储指令、数据和地址等信息。
  - 寄存器的访问速度比内存快得多，但容量较小。
  - 常见的寄存器类型包括：通用寄存器（GPRs）、指令寄存器（IR）、程序计数器（PC）等。

常见的寄存器（以x86-64架构为例）：

- `rax` / `eax` / `ax` / `al`：通用计算寄存器
- `rbx` ： 基址寄存器
- `rcx` ： 计数寄存器
- `rdx` ： 数据寄存器
- `rsi` / `rdi` ： 源变址寄存器
- `rsp` ： 栈顶指针寄存器
- `rbp` ： 栈基址指针寄存器
- `rip` ： 指令指针寄存器（程序计数器）

## 1.2 指令

`指令` 是CPU执行的基本操作单元，每条指令由一个操作码（opcode）和零个或多个操作数组成。
  - 操作码指定要执行的操作，如加法、减法、跳转等。
  - 操作数可以是寄存器、内存地址或立即数。

常见的指令类型：

- `mov` ：赋值（搬运数据）
- `add` ：加法
- `sub` ：减法
- `mul` ：乘法
- `div` ：除法
- `jmp` ：无条件跳转
- `je` / `jne` ： 条件跳转（等于/不等于）
- `call` ：调用函数
- `ret` ：返回函数
- `push` / `pop` ：入栈/出栈
- `cmp` ：比较两个操作数

## 1.3 操作数

操作数是指令中参与运算的对象，可以是寄存器、内存地址或立即数。
  - 寄存器操作数：直接使用寄存器名称，如 `rax`、`rbx` 等。
  - 内存操作数：使用方括号表示内存地址，如 `[rax]` 表示访问 `rax` 寄存器指向的内存地址。
  - 立即数操作数：直接使用数值，如 `5`、`0x10` 等。

# 2. 汇编语言示例

## 2.1 mov 指令示例

```assembly
mov rax, 5        ; rax = 5
mov rbx, rax      ; rbx = rax (rbx = 5)
```

## 2.2 add 指令示例

```assembly
mov rax, 5        ; rax = 5
mov rbx, 10       ; rbx = 10
add rax, rbx      ; rax = rax + rbx (rax = 15)
```

## 2.3 jmp 指令示例

```assembly
start:
    ; some code
    jmp start       ; 无条件跳转回 start 标签
```

## 2.4 cmp 和 je 指令示例

```assembly
mov rax, 5        ; rax = 5
cmp rax, 5        ; 比较 rax 和 5
je equal          ; 如果 rax == 5，跳转到 equal 标签
; some code for not equal case
equal:
    ; some code for equal case
```

## 2.5 call 和 ret 指令示例

```assembly
my_function:
    ; function code
    ret             ; 返回调用者
main:
    call my_function ; 调用 my_function
    ; some code after function call
```

# 3. 栈操作

栈是一种后进先出（LIFO）的数据结构，用于存储函数调用的局部变量、返回地址等信息。
  - `push` 指令将数据压入栈顶，`pop` 指令将数据从栈顶弹出。

```assembly
push rax          ; 将 rax 的值压入栈顶
pop rbx           ; 从栈顶弹出一个值到 rbx
```
