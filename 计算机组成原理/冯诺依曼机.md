
**`存储程序`**：将指令以二进制代码的形式事先输入计算机的**主存储器**，然后按其在存储器的地址顺序逐条取出指令并执行。

# 1. 各个设备

![[img/4/9.png]]

1. **输入设备**：将信息转化成计算机能够识别的形式输入计算机的设备，如键盘、鼠标、扫描仪等。
2. **输出设备**：将计算机处理的结果以人类能够理解的形式输出的设备，如显示器、打印机等。
3. **存储器**：用于存储数据和程序的设备，分为主存储器（RAM）和辅助存储器（如硬盘、U盘等）。
    - 主存储器：直接与CPU进行数据交换，速度较快，但容量有限，断电后数据会丢失。
    - 辅助存储器：容量较大，速度较慢，断电后数据不会丢失，常用于长期存储数据和程序。
4. **运算器**：执行算术和逻辑运算的设备。
5. **控制器**：负责协调和控制计算机各个部件工作的设备。

# 2. 特点

1. 计算机由五大部件组成：输入设备、输出设备、存储器、运算器和控制器。
2. 指令和数据**以同等地位**存储在主存储器中，可以按地址寻访。
3. 指令和数据用二进制表示。
4. 指令由**操作码**和**操作数**两部分组成。
    - 操作码指示要执行的操作
    - 操作数提供执行操作所需的数据。
5. 存储程序。
6. 以运算器为中心，控制器协调各部件工作。
    - 输入/输出设备与存储器之间的数据传送通过运算器完成。

# 3. 现代计算机的结构

![[img/4/10.png]]

与冯诺依曼计算机相比，现代计算机以存储器为中心。

注：
1. 我们熟知的`CPU（中央处理器）`是运算器和控制器的组合体，负责执行指令和协调计算机各部件的工作。
2. **主机**：这里我们指CPU和主存储器的组合体。
    - 主存储器即我们通常说的运行内存（RAM），是计算机的核心存储设备，直接与CPU进行数据交换。
    - 辅存储器（硬盘等）被归类为I/O设备，而不是主机的一部分。
