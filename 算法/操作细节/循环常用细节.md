
# 循环次数的控制

当我想循环 n 次时，可以有如下写法：
```cpp
for(int i = 0; i < n; i++)
{
    // do something
}

for(int i = 1; i <= n; i++)
{
    // do something
}

while(n--)            // 注意这种方法循环后会使 n 变为 0
{
    // do something
}
```

# 用循环来遍历链表

^42ec19

给定如下链表的定义：
```cpp
const int N = 1e6 + 10;
int e[N];
int ne[N];
```

可以选取 0 为哨兵位，不储存数据，进行遍历：
```cpp
for(int cur = ne[0]; cur != 0; cur = ne[cur])
{
    // do something
}

int cur = 0;
while(cur = ne[cur])
{
    // do something
}
```

可以不设置哨兵位，给定头结点 `head` ，进行遍历：
```cpp
for(int cur = head; cur != 0; cur = ne[cur])
{
    // do something
}

int cur = head;
while(cur)
{
    // do something
    
    // 更新 cur
    cur = ne[cur]
}
```
