
首先明确一个事情，** `x % m` 后的结果的范围是 `[0, m-1]` **

当我的范围本身就是 `[0, m-1]` 时候，可以用 `y = (x + 1) % m` 获得 x 的下一位
- 当 `x == m - 1` 时的下一位为 0

当我的范围是 `[1, m]` 时候，可以用 `y = x % m + 1` 获得 x 的下一位
- 当 `x == m` 时的下一位为 1

核心思想为：**把一个长度为 m 的区间平移到 `[0, m]` 的位置，执行完对应的取模运算后，结果还是 `[0, m]` 的区间，再平移回来**

因此，当我要获得 x 的下 k 位（ 1 <= k <= m-1）时可以有如下写法：
- `[0, m-1]`：`y = (x + k) % m`
- `[1, m]`：`y = (x - 1 + k) % m + 1`
- `[n, m-1+n]`：`y = (x - n + k) % m + n`
