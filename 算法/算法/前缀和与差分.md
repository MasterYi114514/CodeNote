
# 1. 前缀和

# 1.1 一维前缀和

可以通过以下的 `init` 函数来创建前缀和数组：

```cpp
const int N = 1e6 + 10;
int n;
int a[N];           // 原数组，有效下标为 1 ~ n
int sum[N];         // 前缀和数组，如果和会超出 int 范围，可以使用 long long 类型

void init()
{
    for(int i = 1; i <= n; i++)
    {
        sum[i] = sum[i - 1] + a[i];
    }
}
```

**实现区间求和**

通过前缀和的预处理后，我们可以做到在 O(1) 的时间内求出任意区间 [l, r] 的和：

```cpp
int query(int l, int r)
{
    return sum[r] - sum[l - 1];
}
```

当然，如果有了 `sum` 数组后，可以通过 `sum[i] - sum[i - 1]` 来得到原数组中的元素 `a[i]`，因此无需存储原数组 `a`，直接在输入时就将元素累加到 `sum` 数组中：

```cpp
for(int i = 1; i <= n; i++)
{
    cin >> sum[i];
    sum[i] += sum[i - 1];
}
```

# 1.2 二维前缀和

二维前缀和总体与一维前缀和类似，区别在于需要考虑两个维度。可以通过以下的 `init` 函数来创建二维前缀和数组：

```cpp
const int N = 1e3 + 10;
int n, m;
int a[N][N];       // 原数组，有效下标为 1 ~ n, 1 ~ m
int sum[N][N];     // 二维前缀和数组，如果和会超出 int 范围，可以使用 long long 类型

void init()
{
    for(int i = 1; i <= n; i++)
    {
        for(int j = 1; j <= m; j++)
        {
            sum[i][j] = sum[i - 1][j] + sum[i][j - 1] - sum[i - 1][j - 1] + a[i][j];
        }
    }
}
```

**实现范围矩阵求和**

通过二维前缀和的预处理后，我们可以做到在 O(1) 的时间内求出任意矩形区域 [x1, y1] ~ [x2, y2] 的和：

```cpp
int query(int x1, int y1, int x2, int y2)
{
    return sum[x2][y2] - sum[x1 - 1][y2] - sum[x2][y1 - 1] + sum[x1 - 1][y1 - 1];
}
```

# 2. 差分

# 2.1 一维差分

差分可以看作是前缀和的逆过程。对于一维差分，我们可以通过以下的 `init` 函数来创建差分数组：

```cpp
const int N = 1e6 + 10;

int n;
int a[N];       // 原数组，有效下标为 1 ~ n
int diff[N];    // 差分数组

void init()
{
    diff[1] = a[1];
    for(int i = 2; i <= n; i++)
    {
        diff[i] = a[i] - a[i - 1];
    }
}
```

**实现区间加法**

通过差分的预处理后，我们可以做到在 O(1) 的时间内对任意区间 [l, r] 进行加法操作：

```cpp
void update(int l, int r, int val)
{
    diff[l] += val;
    if(r + 1 <= n) diff[r + 1] -= val;
}
```

**查询原数组元素**

当需要查询原数组中的元素时，可以通过差分数组来恢复：

```cpp
int query(int i)
{
    int ret = 0;
    for(int j = 1; j <= i; j++)
    {
        ret += diff[j];
    }
    return ret;
}
```

该查询操作的时间复杂度为 O(n)，如果需要频繁查询原数组中的元素，效率较低。

因此，效率更高的方法是通过前缀和的方式直接恢复整个原数组：
```cpp
int a[N];           // 原数组
void build()
{
    // 通过差分数组来恢复原数组
    for(int i = 1; i <= n; i++)
    {
        a[i] = a[i - 1] + diff[i];      // 考虑到 a[0] = 0
    }
}
```

# 2.2 二维差分

二维差分的原理与一维差分类似，只不过需要考虑两个维度。我们可以通过以下的 `init` 函数来创建二维差分数组：

```cpp
const int N = 1e3 + 10;

int n, m;
int a[N][N];       // 原数组，有效下标为 1 ~ n, 1 ~ m
int diff[N][N];    // 二维差分数组

void init()
{
    for(int i = 1; i <= n; i++)
    {
        for(int j = 1; j <= m; j++)
        {
            diff[i][j] = a[i][j] - a[i - 1][j] - a[i][j - 1] + a[i - 1][j - 1];
        }
    }
}
```

**实现范围矩阵加法**

通过二维差分的预处理后，我们可以做到在 O(1) 的时间内对任意矩形区域 [x1, y1] ~ [x2, y2] 进行加法操作：

```cpp
void update(int x1, int y1, int x2, int y2, int val)
{
    diff[x1][y1] += val;
    if(y2 + 1 <= m) diff[x1][y2 + 1] -= val;
    if(x2 + 1 <= n) diff[x2 + 1][y1] -= val;
    if(x2 + 1 <= n && y2 + 1 <= m) diff[x2 + 1][y2 + 1] += val;
}
```

**查询原数组元素**

当需要查询原数组中的元素时，可以通过差分数组来恢复：

```cpp
int query(int x, int y)
{
    int ret = 0;
    for(int i = 1; i <= x; i++)
    {
        for(int j = 1; j <= y; j++)
        {
            ret += diff[i][j];
        }
    }
    return ret;
}

int a[N][N];       // 原数组
void build()
{
    // 通过差分数组来恢复原数组
    for(int i = 1; i <= n; i++)
    {
        for(int j = 1; j <= m; j++)
        {
            a[i][j] = a[i - 1][j] + a[i][j - 1] - a[i - 1][j - 1] + diff[i][j];
        }
    }
}
```
