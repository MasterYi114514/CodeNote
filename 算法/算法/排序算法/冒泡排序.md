冒泡排序(bubble sort)是一种简单直观的排序算法,它的工作原理是通过重复地遍历待排序序列,比较相邻元素并交换顺序错误的元素，使得较大的元素逐渐**浮**到序列的顶端。

# 1. 算法实现

基本流程：
1. 从序列的第一个元素开始，依次比较相邻的两个元素。
2. 如果前一个元素比后一个元素大，则交换它们的位置。
3. 重复上述步骤，直到整个序列有序。

```cpp
const int N = 1e6 + 10;

int n;          // 数组长度
int a[N];       // 待排序数组,有效下标 [1, n]

void bubble_sort()
{
    for(int i = 1; i <= n; i++)
    {
        // 此时 [n-i+1, n] 已经是有序的了

        for(int j = 1; j <= n - i; j++)
        {
            if(a[j] > a[j + 1])
                swap(a[j], a[j + 1]);
        }
    }
}
```

# 2. 算法的优化思路

在冒泡排序的过程中，如果某一轮没有发生任何交换，说明序列已经是有序的了，可以提前结束排序过程。

```cpp
void bubble_sort(int *arr, int n)
{
    for(int i = 1; i < n; i++)
    {
        bool swapped = false;  // 标记本轮是否发生交换

        for(int j = 1; j <= n - i; j++)
        {
            if(arr[j] > arr[j + 1])
            {
                swap(arr[j], arr[j + 1]);
                swapped = true;  // 发生了交换
            }
        }

        // 如果本轮没有发生交换,说明数组已经有序,提前结束排序
        if(!swapped)
            break;
    }
}
```

# 3. 算法分析
**时间复杂度**：最坏情况下（当输入数组是逆序的）为 O(n^2)，平均情况下也是 O(n^2)，但在最佳情况下（当输入数组已经是有序的）为 O(n)。
**空间复杂度**：O(1)，因为冒泡排序是原地排序算法，不需要额外的存储空间。
