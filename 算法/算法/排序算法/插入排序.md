插入排序（insertion sort）是一种简单直观的排序算法。它的工作原理是通过构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。

# 1. 算法实现

基本流程：
1. 从第一个元素开始，该元素可以认为已经被排序。
2. 取出下一个元素，在已经排序的元素序列中从后向前扫描。
3. 如果该元素（已排序）大于新元素，将该元素移到下一个位置。
4. 重复步骤3，直到找到已排序的元素小于或者等于新元素的位置。
5. 将新元素插入到该位置后。

```cpp
const int N = 1e6 + 10;

int n;          // 数组长度
int a[N];       // 待排序数组,有效下标 [1, n]

void insertion_sort()
{
    for(int i = 2; i <= n; i++)
    {
        // 此时，a[1,2,..., i-1] 已经是有序的了
        int key = a[i];     // 待插入的元素

        int j = i - 1;
        while(j >= 1 && a[j] > key)
        {
            a[j + 1] = a[j];
            j--;
        }
        a[j + 1] = key;     // 插入元素
    }
}
```

# 2. 算法分析

**时间复杂度**：最坏情况下（当输入数组是逆序的）为 O(n^2)，平均情况下也是 O(n^2)，但在最佳情况下（当输入数组已经是有序的）为 O(n)。
**空间复杂度**：O(1)，因为插入排序是原地排序算法，不需要额外的存储空间。
