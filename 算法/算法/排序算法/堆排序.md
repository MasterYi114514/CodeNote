堆排序(heap sort)是一种基于堆数据结构的比较排序算法，它的工作原理是将待排序序列构建成一个大根堆(或小根堆)，然后依次将堆顶元素与堆的最后一个元素交换位置，并调整堆结构，直到所有元素均排序完毕。

# 1. 算法实现

基本流程：
1. 构建大根堆：将待排序数组构建成一个大根堆，使得每个父节点都大于或等于其子节点。
2. 交换堆顶元素与最后一个元素：将堆顶元素（最大元素）与堆的最后一个元素交换位置。
3. 调整堆结构：将交换后的堆顶元素调整到正确的位置，以保持堆的性质。
4. 重复步骤2和3，直到堆中只剩一个元素。

代码实现如下：
```cpp
const int N = 1e6 + 10;

int n;
int a[N];       // 堆数组

void adjust_down(int parent, int len)
{
    int child = parent * 2;     // 左孩子结点

    while(child <= len)
    {
        if(child + 1 <= len && a[child + 1] > a[child]) 
            child++;

        if(a[parent] >= a[child])
            break;

        swap(a[parent], a[child]);
        parent = child;
        child = parent * 2;
    }
}

void heap_sort()
{
    // 构建大根堆
    for(int i = n / 2; i >= 1; i--)
        adjust_down(i, n);

    // 依次将堆顶元素与最后一个元素交换位置，并调整堆结构
    for(int i = n; i > 1; i--)
    {
        swap(a[1], a[i]);
        adjust_down(1, i - 1);
    }
}
```

# 2. 算法分析

**时间复杂度**：堆排序的时间复杂度为O(n log n)，其中构建堆的时间复杂度为O(n)，每次调整堆的时间复杂度为O(log n)，总共需要调整n-1次。
**空间复杂度**：堆排序的空间复杂度为O(1)，因为它是原地排序算法，不需要额外的存储空间。
