# 方法一： 正序遍历

需要额外空间，将 nums1 中的有效元素拷贝到临时数组中，然后进行归并。

这里选择了将 nums1 的有效元素拷贝到了 nums1 的后半部分，节省了空间，但是没有去除 O(m) 的额外时间开销。

```cpp
class Solution 
{
public:
    void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) 
    {
        int cur1 = nums1.size(), cur2 = 0;
        for(int i = m - 1; i >= 0; i--)     // 将 nums1 中的有效元素后移
        {
            nums1[--cur1] = nums1[i];
        }

        int cur = 0;
        while(cur1 < nums1.size() && cur2 < nums2.size())
        {
            if(nums1[cur1] <= nums2[cur2])
            {
                nums1[cur++] = nums1[cur1++];
            }
            else
            {
                nums1[cur++] = nums2[cur2++];
            }
        }

        // 如果 nums2 还有剩余，全部拷贝到 nums1 中
        while(cur2 < nums2.size())
        {
            nums1[cur++] = nums2[cur2++];
        }
    }
};
```

# 方法二： 逆序遍历

如果从后向前遍历，相当于已经实现了 `nums1` 的元素后移的操作。

```cpp
class Solution 
{
public:
    void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) 
    {
        int cur1 = m - 1, cur2 = n - 1;
        int cur = nums1.size() - 1;

        while(cur1 >= 0 && cur2 >= 0)
        {
            if(nums1[cur1] >= nums2[cur2])
            {
                nums1[cur--] = nums1[cur1--];
            }
            else
            {
                nums1[cur--] = nums2[cur2--];
            }
        }

        // 如果 nums2 还有剩余，全部拷贝到 nums1 中
        while(cur2 >= 0) nums1[cur--] = nums2[cur2--];
    }
};
```
