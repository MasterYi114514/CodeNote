# 1. 并查集的表示

## 1.1 双亲表示法

可以用以下代码实现树的双亲表示法：

```cpp
const int N = 1e6 + 10;

int n;
int a[N];
int father[N];
```

其中 `father[i]` 表示元素 `i` 的父节点。

这种方法可以在 `O(1)` 的时间内找到元素 `i` 的父节点。

## 1.2 并查集的表示

并查集的实现是用双亲表示法实现的森林。

```cpp
const int N = 1e6 + 10;

int n;
int a[N];
int father[N];
```

**初始化**

并查集中，我们先将每个元素的父节点初始化为自己：

```cpp
void init()
{
    for(int i = 1; i <= n; i++)
    {
        father[i] = i;
    }
}
```

在以后的操作中，我们用父节点等于自己的元素来作为集合的代表元。

**查询操作**

```cpp
int find(int x)
{
    if(father[x] == x) return x;
    return father[x] = find(father[x]);     // 压缩路径，优化查询效率
}
```

或者可以简写为：

```cpp
int find(int x)
{
    return father[x] == x ? x : father[x] = find(father[x]);
}
```

**合并操作**

```cpp
void union_set(int a, int b)
{
    int fa = find(a);
    int fb = find(b);
    if(fa != fb)
    {
        father[fa] = fb;     // 将 a 的代表元连接到 b 的代表元上
    }
}
```
