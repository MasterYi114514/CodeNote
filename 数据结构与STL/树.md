# 1. 树的定义和基本性质

树是一种数据结构，由节点和边组成。树具有以下基本性质：
1. 树是一个无环连通图。
2. 树的节点数为n，则树的边数为n-1。
3. 树的高度是指从根节点到叶子节点的最长路径上的边数。


## 1.1 常见术语

- 根节点：树的顶端节点，没有父节点。
- 父节点与子节点：在树中，如果节点A是节点B的父节点，那么节点B就是节点A的子节点。
- 叶子节点：没有子节点的节点。
- 内部节点：至少有一个子节点的节点。
- 兄弟节点：具有相同父节点的节点。
- 祖先节点：在树中位于某个节点到根节点的路径上的所有节点。
- 后代节点：在树中位于某个节点到叶子节点的路径上的所有节点。
- 结点的度：一个节点的子节点数量。

## 1.2 树的分类

### 1.2.1 有序树和无序树

- 有序树：在有序树中，节点的子节点是有顺序的，即子节点之间存在先后关系。
- 无序树：在无序树中，节点的子节点没有顺序关系。

### 1.2.2 有根树和无根树

- 有根树：有根树是指树中存在一个特殊的节点，称为根节点，所有其他节点都可以通过根节点访问到。
- 无根树：无根树是指树中没有根节点，所有节点之间都是平等的，没有层次关系（可以选取任意的节点作为“根”节点）。

# 2. 树的常见表示方法

## 2.1 邻接表表示法

**邻接表表示法**：我们存储与每一个节点相邻的所有节点的列表。

```cpp
const int N = 1e5 + 10;         // 假设树的最大节点数为N
vector<int> edges[N];           // 邻接表，edges[i] 存储与节点i相邻的所有节点
```

比如我们获取的树的一条边是 `(u, v)`，我们可以将 v 添加到 u 的邻接表中，同时将 u 添加到 v 的邻接表中：

```cpp
edges[u].push_back(v);
edges[v].push_back(u);
```

## 2.2 链式前向星

**链式前向星**：链式前向星是一种特殊的邻接表表示法，使用数组来存储边的信息。

这里我们用静态数组来储存有关信息：
```cpp
const int N = 1e5 + 10;         // 假设树的最大节点数为N
int head[N];                    // head[i] 存储节点i的第一条边的索引
int e[2 * N], ne[2 * N];        // e[i] 存储第i条边的终点，ne[i] 存储第i条边的下一条边的索引
int idx;                        // 当前边的索引
```

我们可以通过以下函数来添加边：

```cpp
void add(int u, int v)
{
    id++;
    e[idx] = v;          // 存储边的终点
    ne[idx] = head[u];   // 将当前边的下一条边指向节点u的第一条边
    head[u] = idx;       // 更新节点u的第一条边的索引为当前边的索引
}
```

# 3. 树的遍历

树的遍历是指按照一定的顺序访问树中的每个节点。常见的树的遍历方式有以下几种：

## 3.1 深度优先遍历（DFS）

深度优先遍历（DFS）是一种递归的遍历方式，按照以下顺序访问节点：
1. 访问当前节点。
2. 递归访问当前节点的每个子节点。
3. 回退到父节点继续访问其他子节点。

```cpp
int n;                      // 树的节点数
vector<int> edges[N];       // 邻接表表示树, edges[i][j] 存储节点i的第j个子节点
bool visited[N];            // 访问标记数组, 访问过的节点标记为 true

void dfs(int u)
{
    // 1. 访问当前节点
    visited[u] = true;
    cout << u << " ";       // 输出当前节点, 或按需要执行其他操作

    // 2. 递归访问当前节点的每个子节点
    for(auto v : edges[u])
    {
        if(visited[v]) continue;        // 调过已经访问过的节点

        dfs(v);                         // 递归访问子节点
    }
}
```

时间复杂度：O(n)，其中n是树的节点数。每个节点被访问一次。
空间复杂度：O(n)，最坏情况下树的高度为n，递归调用栈的空间复杂度为O(n)。

## 3.2 广/宽度优先遍历（BFS）

宽度优先遍历，又称为层序遍历，是一种非递归的遍历方式，按照以下顺序访问节点：
1. 访问当前节点。
2. 将当前节点的所有子节点加入队列。
3. 从队列中取出下一个节点，重复步骤1和2，直到队列为空。

BFS的遍历过程如下：
```
初始队列加入根结点A：[ A ]

访问结点A，加入子节点 BCD： [ A B C D ]
A出队： [ B C D ]       // 这时候剩下的结点就是第二层的全部节点

访问结点B，加入子节点 E F： [ B C D E F ]
B出队： [ C D E F ]
访问结点C，加入子节点 G： [ C D E F G ]
C出队： [ D E F G ]
访问结点D，没有子节点
D出队： [ E F G ]       // 这时候剩下的结点就是第三层的全部节点

访问结点E，没有子节点
E出队： [ F G ]
访问结点F，子节点 H： [ F G H ]
F出队： [ G H ]
访问结点G，没有子节点
G出队： [ H ]           // 这时候剩下的结点就是第四层的全部节点

访问结点H，没有子节点
H出队： [ ]             // 这时候队列为空，遍历结束
```

代码实现：
```cpp
int n;                      // 树的节点数
vector<int> edges[N];       // 邻接表表示树, edges[i][j] 存储节点i的第j个子节点
bool visited[N];            // 访问标记数组, 访问过的节点标记为 true

void bfs()
{
    // 1. 初始化队列并加入根节点
    queue<int> q;
    q.push(1);                  // 假设根节点编号为1，事实上树的根节点可以是任意一个节点，根据实际情况调整
    visited[1] = true;          // 标记根节点为已访问

    // 2. 循环访问队列中的节点，直到队列为空
    while(q.size())             // 或： while(!q.empty())
    {
        int u = q.front();      // 获取队列头部节点
        q.pop();                // 将队列头部节点出队

        cout << u << " ";       // 输出当前节点, 或按需要执行其他操作

        // 3. 将当前节点的所有子节点加入队列
        for(auto v : edges[u])
        {
            if(visited[v]) continue;        // 跳过已经访问过的节点

            visited[v] = true;              // 标记子节点为已访问
            q.push(v);                      // 将子节点加入队列
        }
    }
}
```

时间复杂度：O(n)，其中n是树的节点数。每个节点被访问一次。
空间复杂度：O(n)，最坏情况下树的宽度为n/2，队列中可能同时存在n/2个节点，因此空间复杂度为O(n)。
