# 1. 基础概念

`二叉树` 是一种特殊的树数据结构，其中每个节点**最多有两个子节点**，分别称为 `左子节点` 和 `右子节点`。二叉树具有以下基本性质：
1. 二叉树的子树有左右之分，不可以任意颠倒次序（因此二叉树是有序树的一种特殊情况）。
2. 第i层最多有2^(i-1)个节点。
3. 二叉树的左右子树也都是二叉树。

## 1.1 满二叉树

`满二叉树` 是指每个非叶子结点均有两个孩子，且所有叶子节点都在同一层上的二叉树。

满二叉树具有以下性质：
1. 满二叉树的节点数为2^h - 1，其中h是树的高度。
2. 满二叉树的叶子节点数为2^(h-1)。
3. 将满二叉树按照层序遍历的过程编号，根节点从1开始编号，则：
   - 结点 i 的左子节点编号为 2*i，右子节点编号为 2*i + 1。
   - 结点 i 的父节点编号为 i/2。

## 1.2 完全二叉树

`完全二叉树`是指除最后一层外，每层节点数都达到最大，并且最后一层的节点都集中在左侧的二叉树。

可以理解为 `完全二叉树` 是 `满二叉树` 从后往前删掉部分叶子结点的结果。

# 2. 二叉树的存储

## 2.1 顺序存储

顺序存储是指将二叉树的节点按照层序遍历的顺序存储在数组中。对于一个节点i：
- 左子节点的索引为2*i
- 右子节点的索引为2*i + 1
- 父节点的索引为i/2

```cpp
const int N = 1e5 + 10;         // 假设树的最大节点数为N
int tree[N];                    // 顺序存储的二叉树数组

int lc(int i) { return 2 * i; }         // 获取左子节点索引
int rc(int i) { return 2 * i + 1; }     // 获取右子节点索引
int parent(int i) { return i / 2; }     // 获取父节点索引
```

## 2.2 链式存储

```cpp
struct Node
{
    int val;            // 节点的值
    Node* left;         // 左子节点指针
    Node* right;        // 右子节点指针

    Node(int x) : val(x), left(nullptr), right(nullptr) {} // 构造函数
}

Node* root = nullptr; // 二叉树的根节点指针

// 示例：创建一个简单的二叉树
root = new Node(1);          // 创建根节点
root->left = new Node(2);    // 创建左子节点
root->right = new Node(3);   // 创建右子节点
```

# 3. 二叉树的遍历

二叉树的遍历是指按照一定的顺序访问二叉树中的每个节点。常见的遍历方式有三种：前序遍历、中序遍历和后序遍历。

## 3.1 前序遍历：先访问根节点，再访问左子树，最后访问右子树。

```cpp
const int N = 1e6 + 10;

int n;
int lc[N], rc[N];           // 存储二叉树的左子节点和右子节点
bool visited[N];           // 记录节点是否被访问过

void dfs_f(int u)
{
    if(u == 0) return;     // 如果节点为0，表示没有子节点，直接返回

    cout << u << ' ';     // 访问当前节点
    visited[u] = true;   // 标记当前节点为已访问

    dfs_f(lc[u]);        // 访问左子树
    dfs_f(rc[u]);        // 访问右子树
}
```

## 3.2 中序遍历：先访问左子树，再访问根节点，最后访问右子树。

```cpp
void dfs_m(int u)
{
    if(u == 0) return;     // 如果节点为0，表示没有子节点，直接返回

    dfs_m(lc[u]);        // 访问左子树

    cout << u << ' ';     // 访问当前节点
    visited[u] = true;   // 标记当前节点为已访问

    dfs_m(rc[u]);        // 访问右子树
}
```

## 3.3 后序遍历：先访问左子树，再访问右子树，最后访问根节点。

```cpp
void dfs_l(int u)
{
    if(u == 0) return;     // 如果节点为0，表示没有子结点

    dfs_l(lc[u]);        // 访问左子树
    dfs_l(rc[u]);        // 访问右子树

    cout << u << ' ';     // 访问当前节点
    visited[u] = true;   // 标记当前节点为已访问
}
```
