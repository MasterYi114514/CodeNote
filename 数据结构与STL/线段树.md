# 1. 线段树基础

线段树是一种二叉树数据结构，主要用于处理区间查询和区间更新问题。它可以在 O(log n) 的时间复杂度内完成这些操作：
- **区间查询**：查询某个区间内的某个属性（如最小值、最大值、和等）。
- **区间更新**：更新某个区间内的值（如增加、设置等）。

## 1.1 线段树的模拟实现

**线段树的构建**

```cpp
const int N = 1e5 + 10;

int lc(int x) { return x << 1; }        // 左子节点
int rc(int x) { return x << 1 | 1; }    // 右子节点

int a[N];           // 原始数组
struct Node
{
    int l, r;       // 区间 [l, r]
    int sum;        // 存储区间属性（如最小值、最大值、和等）,这里以求和为例
}tree[N << 4];      // 线段树数组，需要大小为 4N

// 通过子节点的属性更新父节点的属性
void push_up(int p);

// 递归构建线段树
void build(int p, int l, int r)
{
    tree[p].l = l, tree[p].r = r;

    // 对叶子结点进行初始化
    if(l == r)
    {
        tree[p].sum = a[l];
        return;
    }

    int mid = (l + r) >> 1;
    build(lc(p), l, mid);        // 构建左子树，[l, mid]
    build(rc(p), mid + 1, r);    // 构建右子树，[mid + 1, r]

    // 通过子节点获得父节点的对应属性
    push_up(p);
}
```

按照实际线段树的需要，我们可以有不同的 `push_up` 的函数实现，如：

1. **求和线段树**：

```cpp
// 父节点的 sum 是左右子节点 sum 的和
void push_up(int p)
{
    tree[p].sum = tree[lc(p)].sum + tree[rc(p)].sum;
}
```

2. **最小值线段树**：

```cpp
// 父节点的 min 是左右子节点 min 的较小值
void push_up(int p)
{
    tree[p].min = min(tree[lc(p)].min, tree[rc(p)].min);
}
```

最大值线段树与之类似。

**区间查询**

```cpp
// 这里以求和为例
int query(int p, int l, int r)
{
    // 在调用该函数的时候已经可以确保二者有交集，因此不需要进行相关判断
    // if(tree[p].l > r || tree[p].r < l) return 0;

    // p 对应的区间完全在 [l, r] 内
    if(l <= tree[p].l && tree[p].r <= r) return tree[p].sum;

    int mid = (tree[p].l + tree[p].r) >> 1;

    int ret = 0;    // 根据实际需求初始化返回值，如求和为 0，求最小值为 INF 等

    // 递归查询有交集的子区间
    if(l <= mid) ret += query(lc(p), l, r);        // 查询左子树
    if(r > mid) ret += query(rc(p), l, r);         // 查询右子树

    return ret;
}
```

**单点修改**

```cpp
void push_up(int p);

// 以将 pos 位置的值修改为 val 为例，线段树为求和线段树
void update(int p, int pos, int val)
{
    int l = tree[p].l, r = tree[p].r;

    // 找到叶子节点，进行修改
    if(l == r)
    {
        tree[p].sum = val;
        return;
    }

    int mid = (l + r) >> 1;
    if(pos <= mid) update(lc(p), pos, val);         // 修改左子树
    else update(rc(p), pos, val);                   // 修改右子树

    push_up(p);                 // 通过更新过的孩子调整当前节点
}
```

该操作的时间复杂度为 O(log n)，因为每次递归调用都会将区间分成两半。

**区间修改**

这时如果我们对区间的每个元素都进行修改，那么时间复杂度将会退化为 O(n)，因此我们需要引入**懒标记**来实现区间修改的优化。

```cpp
struct Node
{
    int l, r;           // 区间 [l, r]
    int sum;            // 存储区间属性（如最小值、最大值、和等）,这里以求和为例
    int delta;          // 懒标记，这里以区间增加 delta 为例
}

void push_up(int p);

// 将 p 结点的操作懒下来
void lazy(int p, int delta)
{
    int l = tree[p].l, r = tree[p].r;
    tree[p].sum += (r - l + 1) * delta;   // 根据实际需求进行调整
    tree[p].delta += delta;               // 更新懒标记
}

// 将 p 结点的懒标记下传给子节点
void push_down(int p)
{
    if(!tree[p].delta) return;      // 没有懒标记，直接返回

    lazy(lc(p), tree[p].delta);    // 将懒标记下传给左子节点
    lazy(rc(p), tree[p].delta);    // 将懒标记下传给右子节点
    tree[p].delta = 0;             // 清除当前节点的懒标记
}

// 将区间 [l, r] 内的值增加 delta，以求和线段树为例
void update(int p, int l, int r, int delta)
{
    // 完全在区间内，直接进行懒操作
    if(l <= tree[p].l && tree[p].r <= r)
    {
        lazy(p, delta);
        return;
    }

    int mid = (tree[p].l + tree[p].r) >> 1;

    // 先下放当前懒标记
    push_down(p);

    // 寻找合适的子区间
    if(l <= mid) update(lc(p), l, r, delta);        // 更新左子树
    if(r > mid) update(rc(p), l, r, delta);         // 更新右子树
}
```

注意，在引入懒标记后，执行查询的时候一定要先下放懒标记，以保证查询结果的正确性。

```cpp
int query(int p, int l, int r)
{
    if(l <= tree[p].l && tree[p].r <= r) return tree[p].sum;

    // 先下放当前节点的懒标记
    push_down(p);

    // 递归查询子区间
    int mid = (tree[p].l + tree[p].r) >> 1;
    int ret = 0;    // 根据实际需求初始化返回值，如求和为 0，求最小值为 INF 等
    if(l <= mid) ret += query(lc(p), l, r);        // 查询左子树
    if(r > mid) ret += query(rc(p), l, r);         // 查询右子树

    return ret;
}
```

这样，我们实现的对应的操作：
- 区间查询：O(log n)
- 区间更新：O(log n)
