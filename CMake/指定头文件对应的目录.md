- [[CMake中的变量与set相关用法]]

可以通过 `include_directories` 命令指定头文件要包含的目录：
```cmake
# 用法为 include_directories(头文件所在目录)
# 比如，当前目录为：
# project
# |--- CMakeLists.txt
# |--- include
#      |--- file01.h
#      |--- file02.h
# |--- src
#      |--- main.cpp
include_directories(${CMAKE_CURRENT_SOURCE_DIR}/include)

# 这样，在main.cpp中可以简单包含头文件
# #include <file01.h>
# #include <file02.h>
```
