# 方式一：`aux_source_directory` 命令

使用方法：
```cmake
# aux_source_directory(目录 变量名)
# 比如如下命令实现了查找 D:\test\src 里的所有文件，并存在SRC_LIST中
aux_source_directory(D:\test\src SRC_LIST)
add_executable(hahaha ${SRC_LIST})
```

# 方式二：`file` 命令

基础用法为:
```cmake
file(GLOB/GLOB_RECURSE 变量名 要搜索的文件路径和文件类型)
```

这里第二个参数的选择可以依照：
- `GLOB`：将指定目录里搜索到的满足条件的所有文件名生成一个列表，并存储到变量中
- `GLOB_RECURSE`：对指定目录进行递归搜索，再生成列表、存储在变量中

示例：
```cmake
# 查找src目录中的.cpp文件
file(GLOB MAIN_SRC ${CMAKE_CURRENT_SOURCE_DIR}/src/*.cpp)

# 查找include目录下的.h文件
file(GLOB MAIN_SRC ${CMAKE_CURRENT_SOURCE_DIR}/include/*.h)
```
