# 可执行程序的输出

CMake中有一个预设的宏 `EXECUTABLE_OUTPUT_PATH` ，通过对其赋值即可实现指定输出路径：
```cmake
set(WORKSPACE D:/ABC)
set(EXECUTABLE_OUTPUT_PATH ${WORKSPACE}/bin)
```

注：如果这里的输出路径不存在，CMake会自动生成，无需手动创建
# 库的输出

与可执行程序类似，库文件也有一个预设的宏 `LIBRARY_OUTPUT_PATH` ，这个宏对静态库和动态库都适用，如：
```cmake
add_library(Test SHARED file01.txt file02.txt)
set(LIBRARY_OUTPUT_PATH ${PROJECT_SOURCE_DIR}/lib)
```
