# 库的制作

制作库时使用命令：
```cmake
# 制作静态库
add_library(库名称 STATIC 源文件1 源文件2 源文件3 ...)

# 制作动态库
add_library(库名称 SHARED 源文件1 源文件2 源文件3 ...)
```

例如：
```cmake
cmake_minimun_required(VERSION 4.10)
project(test)

file(GLOB SRC ${CMAKE_CURRENT_SOURCE_DIR}/src/*.cpp)

add_library(test SHARED ${SRC})
```

# 库的链接方法

## 方法一：link_libraries 命令（只适用于静态库）

```cmake
# 库可能由系统提供，也可以能是自己提供的
# 对于后者，需要先用 link_directories命令指定其所在目录
# 该方法对静态库动态库均有效
link_directories(路径1 路径2 ...)

# 在系统路径和上面提供的路径中寻找名字对应的库
# 该命令只用于指定静态库
link_libraries(静态库1 静态库2 静态库3 ...)
```

## 方法二：target_link_libraries 命令（静态库动态库均适用）

```cmake
target_link_libraries
(
	目标文件名    # 需要链接库的文件
	权限(PRIVATE/PUBLIC/INTERFACE) 库1
	权限 库2
	权限 库3
	...
)
```
