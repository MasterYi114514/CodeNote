# 1. 函数的默认参数

在C++中，函数的参数可以有默认值，这样在调用函数时可以省略这些参数。默认参数必须从右向左依次设置。

```cpp
int func(int a, int b = 10, int c = 20) 
{
    return a + b + c;
}

int main() 
{
    int result1 = func(5);          // 使用默认参数 b=10, c=20
    int result2 = func(5, 15);      // 使用默认参数 c=20
    int result3 = func(5, 15, 25);  // 不使用默认参数

    return 0;
}
```

注意:

1. 默认参数只能在函数声明或定义中指定一次。
2. 如果某个参数有默认值，那么它右边的所有参数都必须有默认值。
3. 如果函数声明和定义分开，默认参数应优先在函数声明中指定(也可以在定义中指定，但不能两处都指定)。

# 2. 函数的占位参数

占位参数是指在函数声明或定义中使用但不实际使用的参数。通常用于保持函数签名的一致性。

```cpp
void exampleFunction(int a, int /*b*/, double c) 
{
    // 这里参数 b 没有被使用
    std::cout << "a: " << a << ", c: " << c << std::endl;
}

int main() 
{
    exampleFunction(5, 0, 3.14); // 调用时仍需传递参数 b，但其值不会被使用
    return 0;
}
```

# 3. 函数重载

函数重载是指在同一个作用域内，允许多个函数具有相同的名称，但参数列表（参数类型、参数个数或参数顺序）必须不同。编译器根据调用时传递的参数来决定调用哪个函数。

函数重载需满足以下条件：

1. 函数名相同。
2. 参数列表不同（参数类型、参数个数或参数顺序）。
3. 在同一个作用域内。

```cpp
int add(int a, int b) 
{
    return a + b;
}

double add(double a, double b) 
{
    return a + b;
}

std::string add(const std::string& a, const std::string& b) 
{
    return a + b;
}
```

注意:
1. 函数的返回类型不能作为重载的依据。
2. 函数重载可以与默认参数结合使用，但要注意避免歧义。

## 3.1 const作为重载依据

```cpp
void process(int& a) 
{
    std::cout << "value reference overload" << std::endl;
}

void process(const int& a) 
{
    std::cout << "const value reference overload" << std::endl;
}

int main() 
{
    int x = 10;
    const int y = 20;

    process(x); // 调用第一个重载
    process(y); // 调用第二个重载
    process(30); // 调用第二个重载

    return 0;
}
```

## 3.2 函数重载与默认参数结合使用

当函数重载与默认参数结合使用时，需要确保不会引起歧义。

```cpp
void display(int a, int b = 10) 
{
    std::cout << "a: " << a << ", b: " << b << std::endl;
}

void display(int a) 
{
    std::cout << "a: " << a << std::endl;
}

int main()
{
    display(5);      // 错误写法, 无法确定调用哪个 display 函数
    display(5, 15);  // 调用 display(int a, int b)

    return 0;
}
```
