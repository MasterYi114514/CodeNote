# 1. 基础语法

**类的变长参数模板**

```cpp
template<typename... Args>
class MyClass
{
    // 具体内容
}
```

这里的 `Args` 是一个模版的形参包，可以接受任意数量的类型参数。我们可以在类的定义中使用 `Args...` 来展开这些参数。

**函数的变长参数模板**

```cpp
template<typename... Args>
void func(Args... args)
{
    // 具体内容
}
```

注：
以上的写法也可以表示0个参数的情况，即`MyClass<>`或`func()`均为合法的。
如果希望至少有一个参数，可以使用以下的写法：

```cpp
template<typename T, typename... Args>
class MyClass
{
    // 具体内容
}
```

## 1.1 `sizeof...` 操作符

`sizeof...` 是一个编译时操作符，用于获取参数包中参数的数量。它可以用于类模板和函数模板中，以确定传递了多少个参数。

```cpp
template<typename... Args>
void func(Args... args)
{
    std::cout << sizeof...(args) << std::endl;  // 获取参数数量
}
```

## 1.2 参数包的概念约束

`concept` 是 C++20 引入的一种语言特性，用于定义类型约束。我们可以使用 `concept` 来约束变长参数模板中的类型，确保它们满足特定的条件。

`std::is_arithmetic` 是一个类型特征，用于判断一个类型是否是算术类型（即整数类型或浮点类型）。在使用变长参数模板时，我们可以利用 `std::is_arithmetic` 来约束参数包中的类型，确保它们都是算术类型。

```cpp
#include <type_traits>

// 定义一个概念，约束参数包中的类型必须是算术类型
// 要求 C++20 支持
template<typename T>
concept Arithmetic = std::is_arithmetic_v<T>;

template<Arithmetic... Args>
void func(Args... args)
{
    // 具体内容
}
```

特殊地，如果我们想要参数包中的所有类型都是同一种类型，我们可以使用 `std::same_as` 来约束：

```cpp
#include <iostream>
#include <concepts>

template<std::same_as<int>... Args>
void func(Args... args)
{
    // 示例
    auto s = (args + ... + 0);      // 计算所有参数的和
    std::cout << "Sum: " << s << std::endl;
}
```

# 2. 参数包的展开

## 2.1 递归模板函数

```cpp
#include <iostream>

template<typename T0>
void print(T0 t0)
{
    std::cout << t0 << std::endl;
}

template<typename T, typename... Args>
void print(T t, Args... args)
{
    std::cout << t << std::endl;  // 打印当前参数
    print(args...);               // 递归调用，展开剩余参数
}
```

示例：
```cpp
print(1, 2.5, "Hello");    
```

运行逻辑如下：
```
调用 print(1, 2.5, "Hello"):    
    打印 1，并调用 print(2.5, "Hello")
调用 print(2.5, "Hello"):       
    打印 2.5，并调用 print("Hello")
调用 print("Hello"):            
    打印 "Hello"

前两个调用 print(T t, Args... args)，最后一个调用 print(T0 t0) 的重载并结束递归。
```

输出：
```
1
2.5
Hello
```

## 2.2 变参模板展开

在 C++17 中增加了变参模板展开的支持，使得参数包的展开更加简洁：

```cpp
template<typename T0, typename... Args>
auto print(T0 t0, Args... args)
{
    std::cout << t0 << std::endl;  // 打印当前参数
    if constexpr (sizeof...(args) > 0) 
    {  // 检查是否还有剩余参数
        print(args...);               // 展开剩余参数
    }
}
```

## 2.3 初始化列表展开

初始化列表展开是一种非常简洁的方式来展开参数包，特别适用于需要对每个参数进行相同操作的情况。

以下例子通过 `std::vector` 的初始化列表获得了参数包中所有参数的值，并将它们转换为 `double` 类型：
```cpp
template<std::convertible_to<double>... Args>   // 约束参数必须可转换为 double 类型
void func(Args... args)
{
    std::vector<double> vec{static_cast<double>(args)...};

    for(auto num : vec)
    {
        std::cout << num << " ";
    }

    std::cout << "size of pack: " << sizeof...(args) << std::endl;
}
```

或者也可以借助 `lambda` 表达式来进行展开：

```cpp
template<typename T, typename... Args>
auto print(T t, Args... args)
{
    std::cout << t << std::endl;  // 打印当前参数
    (void)std::initializer_list<int>
    {
        (
            [&args]()
            {
                std::cout << args << std::endl;     // 打印每个参数
            }(), 0
        )...
    };
}
```
