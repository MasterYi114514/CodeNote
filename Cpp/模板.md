模版（Template）是C++中实现泛型编程的一种机制。通过模版，程序员可以编写与类型无关的代码，从而实现代码的重用和灵活性。

# 1. 基本语法

cpp的模版主要分为两种类型：函数模版和类模版。

## 1.1 函数模版

函数模版允许我们定义一个通用的函数，可以处理不同类型的数据。其基本语法如下：
```cpp
template <typename T>
T add(T a, T b) 
{
    return a + b;
}
```

注:
- `template <typename T>`：声明一个模版，`T`是一个类型参数，可以替换为任何数据类型。
- 自动类型推导：在调用函数模版时，编译器会根据传入的参数类型自动推导出`T`的具体类型。
  - 当被推导出一致的类型T时才可以使用函数模版。
  - 当无法推导出类型时,可以显式指定类型参数，例如：`func<int>()`。
- `typename`和`class`关键字在模版声明中是等价的，可以互换使用。

## 1.2 类模版

类模版允许我们定义一个通用的类，可以处理不同类型的数据。其基本语法如下：

```cpp
template <typename T>
class Box 
{
private:
    T value;
public:
    Box(T val) : value(val) {}
    T getValue() { return value; }
};
```

注:
- 类模版的使用方式与函数模版类似，可以通过指定类型参数来创建不同类型的类实例。
  - 例如：`Box<int> intBox(10);` 创建一个存储整数的Box对象。
- 类模版可以有多个类型参数，例如：

```cpp
template <typename T1, typename T2>
class Pair 
{
private:
    T1 first;
    T2 second;
public:
    Pair(T1 f, T2 s) : first(f), second(s) {}
    T1 getFirst() { return first; }
    T2 getSecond() { return second; }
};
```

### 1.2.1 类模板对象做函数参数

类模板对象可以作为函数参数传递。例如：

```cpp
template <typename T>
void printBox(Box<T> box)
{
    std::cout << "Box value: " << box.getValue() << std::endl;
}
```

### 1.2.2 类模版与继承

类模版可以与继承结合使用, 当父类为模版类而子类不是时, 需要指定父类模版的类型参数。

例如：

```cpp
class DrivedBox : public Box<int> 
{
public:
    DrivedBox(int val) : Box<int>(val) {}
};
```

或者,对于子类本身也是模版类, 可以这样写:

```cpp
template <typename T>
class DrivedBox : public Box<T>
{
public:
    DrivedBox(T val) : Box<T>(val) {}
};
```

### 1.2.3 类模版成员函数的类外实现

类模版的成员函数可以在类外实现。需要注意的是，在类外实现时，必须重新指定模版参数列表。例如：

```cpp
int Box<int>::getValue()
{
    return value;
}

template <typename T>
T Box<T>::getValue()
{
    return value;
}
```

# 2. 模版特化

模版特化允许我们为特定类型提供专门的实现。C++支持两种类型的模版特化：完全特化和部分特化。

## 2.1 完全特化

完全特化是为特定类型提供专门的实现。例如：

```cpp
template <>
class Box<int> 
{
private:
    int value;
public:
    Box(int val) : value(val) {}
    int getValue() { return value; }
    void show() { std::cout << "This is an integer box: " << value << std::endl; }
};
```

## 2.2 部分特化

部分特化允许我们为一部分类型提供专门的实现。例如：
```cpp
template <typename T>
class Box<T*> 
{
private:
    T* value;
public:
    Box(T* val) : value(val) {}
    T* getValue() { return value; }
    void show() { std::cout << "This is a pointer box." << std::endl; }
};
```
