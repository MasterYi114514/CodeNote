
# 核心：确保一个类只有一个实例，并且自行实例化，并向整个类提供这个实例

- 构造方法要是 `private` ，不可被外部调用
- 这个实例应该是当前类的静态成员变量（`static`）
- 需要有一个方法，向外界提供当前类的实例

# 饿汉式：在类加载的时候就进行实例化

```cpp
class Singleton
{
public:
	static Singleton& get_instance()
	{
		return instance;
	}
	
private:
	// 禁止拷贝与赋值
	Singleton(const Singleton& other) = delete;
	Singleton& operator=(const Singleton& other) = delete;
	
private:
	Singleton() = default;
	~Singleton() = default;
	
	static Singleton instance;
};

inline Singleton Singleton::instance;
```

# 懒汉式：在第一次使用的时候进行实例化

```cpp
class Singleton
{
public:
	static Singleton& get_instance()
	{
		static Singleton instance;
		return instance;
	}
	
private:
	// 禁止拷贝与赋值
	Singleton(const Singleton& other) = delete;
	Singleton& operator=(const Singleton& other) = delete;
	
private:
	Singleton() = default;
	~Singleton() = default;
};
```
